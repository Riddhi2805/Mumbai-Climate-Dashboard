# 🌍 Mumbai Climate Analytics Platform

An interactive Streamlit-based climate analytics platform designed to analyze historical climate patterns in Mumbai, simulate climate variables using machine learning, explore climate change trends, and perform user-driven dataset analysis.

## 🚀 Live Application

🔗 Streamlit App: https://mumbai-climate-dashboard-h2tfhv76viwj9htk2zpuby.streamlit.app/

## 📌 Project Overview

The Mumbai Climate Analytics Platform helps users understand climate behavior in Mumbai through:

* Historical climate data analysis (2010–2023)
* Interactive climate simulations
* Climate trend monitoring
* Correlation and relationship analysis
* User dataset exploration
* Live weather information integration

The platform combines data visualization, exploratory data analysis (EDA), statistical interpretation, and machine learning to provide meaningful climate insights.

## 🤖 Machine Learning Component

The simulation module utilizes Linear Regression models to estimate climate variables based on selected environmental factors.

Users can interactively adjust predictor variables and observe how the predicted climate parameter changes.
